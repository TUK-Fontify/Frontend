# Frontend (React)
React(Next.js)로 만든 웹 프론트엔드입니다.
현재 메인 페이지, 영문 폰트 상세 페이지, 한글 폰트 상세 페이지가 구현된 상태입니다

## 필요 프로그램
- Node.js (npm 포함)
- 
## 로컬에서 실행하기
1. 저장소 폴더로 이동
   - `cd Frontend`
   - 
2. 의존성 설치
   - `npm install`
   - 
3. 개발 서버 실행
   - `npm run dev`
   - 
4. 브라우저에서 접속
   - `http://localhost:3000`
